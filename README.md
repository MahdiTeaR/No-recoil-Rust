## üö® CRITICAL UPDATE (As of June 2025)

The `Interception` driver, which is essential for this program, is now frequently blocked by major anti-cheat systems. As a result, the program **will not work correctly** under normal circumstances.

A temporary workaround has been found, but it requires careful execution and carries significant risk.

### Workaround & VERY IMPORTANT WARNING

1.  Before launching your game, you must rename the Interception driver files (`keyboard.sys`((to temp.sys)) and `mouse.sys`((to temp1.sys))).
2.  Launch the game.
3.  Once the game is fully running, you **MUST** rename the files back to their original names (`keyboard.sys`, `mouse.sys`).

> [!DANGER]
> **CRITICAL RISK**: IF YOU RESTART YOUR PC WITHOUT RENAMING THE FILES BACK TO THEIR ORIGINAL NAMES, YOUR MOUSE AND KEYBOARD **WILL NOT WORK**. You will need to use advanced recovery methods to fix your system. Proceed with extreme caution.

The `Interception` driver is still required for the program to run. We are actively seeking a better solution.

## üöÄ Getting Started

### Prerequisites

- **Operating System**: Windows 10/11 (64-bit)
- **RAM**: Minimum 4GB
- **Graphics**: DirectX 11 or higher
- **Interception Driver**: You must install the `Interception` driver (included in the first launch).

### Installation

1.  Download the `Core.exe` file.
2.  Place it in a folder of your choice.
3.  Run the application **as an Administrator**.
4.  On the first launch, the `Interception` driver will be installed automatically.
5.  **Restart your system** after the driver installation is complete.
6.  Relaunch the application and follow the instructions in the **CRITICAL UPDATE** section above.

## üéÆ Usage Guide

### Initial Setup

1.  After launching the program, navigate to the **Settings** tab.
2.  Enter your in-game **FOV** and **Sensitivity** values to match your game's settings.
3.  Configure the attachment settings for each weapon as needed.

### Profile Selection

1.  Go to the **Profiles** tab.
2.  Select your desired weapon.
3.  Assign a hotkey for that profile.
4.  Press the assigned hotkey in-game to activate the profile.

### Default Hotkeys

-   `F3` - `F10`: Select different weapon profiles.
-   `Home`: Toggle the user interface (UI) on/off.
-   `Insert`: Exit the application.
-   `END`: Night Mode.
